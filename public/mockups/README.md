# Mockups Directory

This directory contains product mockup images for the design studio.

## File Naming Convention

Format: `{product}-{view}-{color}.png`

Examples:
- `tshirt-front-white.png`
- `tshirt-back-black.png`
- `hoodie-front-gray.png`

## Image Specifications

- **Format**: PNG with transparency
- **Dimensions**: 1000x1200px minimum
- **Resolution**: 72 DPI (web display)
- **Color Mode**: RGB
- **File Size**: < 500KB per mockup

## Print Area Specifications

Each mockup should have a defined print area where designs can be placed:

**T-Shirt Front/Back:**
- Position: Center chest
- Size: 12" x 16" (400x500px at mockup scale)
- Coordinates: x=300, y=200

**T-Shirt Side:**
- Position: Visible side
- Size: 8" x 12" (270x400px at mockup scale)
- Coordinates: x=400, y=250

## Sources

Mockup images can be obtained from:
- **Printful**: Product mockup generator
- **Placeit**: Mockup templates
- **Adobe Stock**: Professional mockups
- **Custom**: Photography or 3D rendering

## Notes

- Ensure mockups have consistent lighting and perspective
- Use neutral backgrounds for better design visibility
- Keep mockup files in version control for consistency across environments
